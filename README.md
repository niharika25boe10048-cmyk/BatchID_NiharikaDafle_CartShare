# CartShare

## Objective

Collaborative shopping cart application for shared purchases.

## Features

- Create Room
- Join Room
- Shared Cart
- Add Item
- Delete Item
- Activity Log
- Real-Time Multi-Tab Updates
- Local Storage Persistence
- Printable Receipt
- Responsive Design

## Technologies

- HTML
- CSS
- Bootstrap
- JavaScript
- LocalStorage

## Run Project

1. Download project
2. Open index.html
3. Create or Join Room
4. Add Items
5. Generate Receipt
